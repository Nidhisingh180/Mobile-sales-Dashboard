📱 Mobile Sales Dashboard
1. Project Title / Headline
📊 Mobile Sales Analytics Dashboard

An interactive Power BI dashboard designed to analyze mobile phone sales performance across different cities, brands, models, payment methods, and customer ratings. The dashboard helps uncover sales trends, customer behavior, and business insights through dynamic visualizations and KPI tracking.

2. Short Description / Purpose

The Mobile Sales Dashboard provides a complete overview of mobile sales data using interactive visuals and KPIs. It enables users to monitor sales performance, identify top-selling products, analyze customer ratings, and understand transaction patterns across multiple cities and time periods.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Main platform used for dashboard creation and data visualization.
📂 Power Query – Used for data cleaning, transformation, and preparation.
🧠 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and dynamic reporting.
🔗 Data Modeling – Relationships created between sales-related tables for interactive filtering and analysis.
📁 File Formats – .pbix for development and .png for dashboard preview images.
4. Data Source

Source: Sample Mobile Sales Dataset

The dataset contains mobile sales transaction details including:

Mobile brands and models
Total sales and quantity sold
Customer ratings
Payment methods
Monthly transactions
City-wise sales distribution

The dataset was structured and transformed to support KPI calculations, trend analysis, and interactive reporting.

5. Features / Highlights
• Business Problem

Mobile retailers generate large amounts of sales data daily, but raw transactional data makes it difficult to quickly identify:

Best-selling mobile models
High-performing cities
Preferred payment methods
Monthly sales trends
Customer satisfaction patterns

Businesses need a centralized analytical dashboard to monitor performance and make data-driven decisions.

• Goal of the Dashboard

The dashboard was designed to:

Track overall mobile sales performance
Analyze customer purchasing behavior
Identify top-performing brands and models
Monitor monthly sales and transaction trends
Compare customer ratings and payment preferences
Provide interactive filtering for better business analysis
